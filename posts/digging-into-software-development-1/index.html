Digging into software development (part 1)

July 20, 2018

Purpose

I always wanted to touch the holistic approach of TTD/BDD with CI/CD So this is the purpose of this series of blog posts. Now I have the oportunity to experiment more than usual. With this blog post I intend to dig into those subjects in the context of the following parameters:

  • Golang as the language of choice
  • Bosh as the deployment tool
  • Concourse as the CI tool

My target is to go even further to the Google ecosystem and touch Kubernetes even. I will also use Cloud Foundry platform and tools as an other means to deploy my software and run everything I want to run. We will see how it goes. While doing that I will try to utilise some other tools and explain my developing process trying to make it better. I am using the following tools to code:

  • tmux
  • neovim
  • my dotfiles
