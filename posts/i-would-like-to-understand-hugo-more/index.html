I don't get hugo as much as I would like to

July 7, 2018

This is my third attempt to use Hugo. I must admit some times trying to persuade it that a post is not a draft is a bit painful. And to be honest I am not sure if it is Hugo’s fault or the theme’s. For what is worth I desided I am going to keep using it. There are a bunch of static site generators and each time I try to check which one I should use I end up on Jekyll. I love Jekyll and github is very happy with it. I just think it was time to move on to something new, God knows why :-P.
